# Jearnell Ken Time Tracker

## Project Description
A browser-based time tracking tool that allows users to add tasks, track time with a running timer, and view daily and weekly summaries. Includes Start, Stop, and Delete functionality for task management.

## Features
- Add tasks with categories.
- Start tasks and view running duration.
- Stop tasks to save end time.
- Delete tasks if added by mistake.
- Dashboard showing total time for today and this week.
- Data is saved in browser localStorage for persistence.

## Tech Stack
- HTML, CSS, JavaScript (No external libraries required).

## How to Use
1. Download or clone the folder.
2. Open `index.html` in your browser.
3. Add tasks using the input box and category selector.
4. Start, stop, or delete tasks.
5. Dashboard updates in real-time.
